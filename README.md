## 📖 Sobre Mí

¡Bienvenidos a mi GitHub! Soy una apasionada por la enseñanza y el desarrollo de software, con un enfoque especial en el lado del servidor (Backend). Mi misión es ayudar a mis estudiantes a comprender la lógica detrás de las aplicaciones modernas y a construir **APIs robustas, escalables y seguras utilizando Node.js**.

* 🔭 **Actualmente enseñando:** Arquitectura de APIs RESTful, Node.js, Express y bases de datos.
* 🌱 **Siempre aprendiendo:** Nuevas tendencias en arquitectura de microservicios y seguridad web.
* 👯 **Dispuesta a colaborar en:** Proyectos educativos, mentorías y código abierto.
* 💬 **Pregúntame sobre:** JavaScript, Node.js, Express.js, MongoDB, SQL y despliegue de aplicaciones.
* ⚡ **Dato curioso:** ¡Encontrar bugs es mi deporte extremo favorito!
